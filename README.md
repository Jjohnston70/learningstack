# learningstack

Operator-grade learning artifacts and starter code for building modular, multi-tenant LLM platforms.

## What This Repo Contains
- `llm-platform-starter-sets/`: Python and TypeScript starter runtimes, module template, and content packs.
- `LLM_PLATFORM_COMPLETE_LEARNING_MODULE.md`: End-to-end architecture learning reference.
- `DECISION ENGINE LEARNING MODULE.md`: Decision engine design and implementation reference.
- `LLM-BUILD-PROMPTS/`: Reusable build and review prompts.

## Why It Exists
This repo demonstrates production-minded patterns for:
- shared-core multi-client architecture
- module-based orchestration
- tenant isolation and redaction-first logging
- staged rollout and governance practices

## Quick Start
### Python starter
```bash
cd llm-platform-starter-sets/python-starter
pip install -r requirements.txt
python -m app.main
```

### TypeScript starter
```bash
cd llm-platform-starter-sets/typescript-starter
npm install
npm run start
```
