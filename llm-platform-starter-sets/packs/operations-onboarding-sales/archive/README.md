# Archive

This folder contains historical versions, older docs, and alignment summaries.

## What belongs here

- Previous versions of workflow docs
- Older proposal or contract templates (superseded)
- Internal planning and alignment summaries
- Migration notes from legacy systems

## What does NOT belong here

- Anything the runtime needs
- Active module source code
- Current template IDs

## Rule

If a file in archive contradicts something in `docs/` or `manifests/`,
the docs/manifests version wins. Archive is reference only.
